Via Hosts File

Add the following to your hosts file to access the running servers.

PUBLIC_IPmaster.lrsupport.commaster
PUBLIC_IP71x.lrsupport.com71x
PUBLIC_IP70x.lrsupport.com70x
PUBLIC_IP62x.lrsupport.com62x
PUBLIC_IP61x.lrsupport.com61x
PUBLIC_IPdebug.lrsupport.comdebug

Via group-server-room

If you are connected to the VPN, you can access the running servers directly.

masterhttp://PUBLIC_HOSTNAME:6080/
71xhttp://PUBLIC_HOSTNAME:7180/
70xhttp://PUBLIC_HOSTNAME:7080/
62xhttp://PUBLIC_HOSTNAME:6280/
61xhttp://PUBLIC_HOSTNAME:6180/
debughttp://PUBLIC_HOSTNAME:8080/

All servers are started with JPDA, so you can also attach your IDE debugger to the servers.

masterPUBLIC_IP, port 6000
71xPUBLIC_IP, port 7100
70xPUBLIC_IP, port 7000
62xPUBLIC_IP, port 6200
61xPUBLIC_IP, port 6100
debugPUBLIC_IP, port 8000
